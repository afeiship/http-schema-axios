# next-fetch
> Abstract for browser or node.

## installation
```bash
npm install -S @feizheng/next-fetch
```

## apis
| api    | params | description   |
| ------ | ------ | ------------- |
| get    | -      | desc balabala |
| post   | -      | desc balabala |
| put    | -      | desc balabala |
| patch  | -      | desc balabala |
| delete | -      | desc balabala |
| head   | -      | desc balabala |

## usage
```js
import NxFetch from '@feizheng/next-fetch';

const http = new NxFetch().getInstance();

// methods list:
http.get
http.post
http.put
http.patch
http.delete
http.head
```
